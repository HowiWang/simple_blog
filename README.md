## 简单的博客论坛

#### 1.介绍
使用前后端分离的形式开发一个简单的博客论坛。
主要是用node.js+express+Vue+ElementUI+mysql创建的项目


#### 2.使用的框架
* 前端
    + Vue-cli脚手架
* 后台
    + node.js express框架


#### 3.完成的主要功能
    登录注册、session登录状态、文章的发表、文章列表渲染、单篇文章展示、富文本框编辑


#### 4.使用

* npm i
* 前端：npm run serve
* 后端：node app
* 将sql文件导入到mysql数据库中，并启动mysql服务
* 打开 localhost:8080
